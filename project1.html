Project 1: Top US Baby Names

#R, #dataviz, #animation

November 1, 2015 Du Pham

This project is my final project in a five-course certificate that I take from Coursera about advanced DataViz with R. I choose the US baby names for this project because the data set has a long temporal variable. I made several SQL queries in SQL workspace in BigQuery of Google Cloud Platform since the original data file has more than 6 million rows, which is very heavy to manipulate in R.

Please check out my work Top US Baby Names Dataviz (may take sometime to load) on Shiny server.
